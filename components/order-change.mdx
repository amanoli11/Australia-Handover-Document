# 💰 Order Change

Manage an Order Change request from submission through approval, cash processing, and route planning.

## 📖 Overview

| Item | Details |
| --- | --- |
| **Module** | Cash Order |
| **Estimated Time** | 5–10 minutes |
| **Workflow** | Customer → Vault Manager → Cash Supervisor → Cash Processor → Route Planner |

## ✅ Before You Begin

- Ensure you have the required permissions.
- Verify the customer account is active.
- Confirm the order is eligible for modification.
- Prepare all required information before starting.

## 🔄 Workflow

```text
Customer
   │
   ▼
Vault Manager
   │
   ▼
Cash Supervisor
   │
   ▼
Cash Processor
   │
   ▼
Route Planner
```
