# Non Cash Item

The **Non Cash Item** feature allows customers to request non-cash items, such as consumables or operational supplies, through the Authentic Online portal. Users can create a new order, review the order details, complete the crew sign-off process, and cancel the order if required.

Once submitted, the order can be assigned to a run by the Route Planner for delivery and tracked throughout its lifecycle.

:::tip
This guide walks you through the complete Non Cash Item workflow, from creating an order to assigning it for delivery.
:::
