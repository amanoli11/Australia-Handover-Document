# Route Planner Process

Once a Non Cash Item Order has been created and confirmed, the **Route Planner** is responsible for assigning the order to a delivery run and generating a Runsheet.

This section explains how to assign orders, create a Runsheet, and monitor the delivery status.

---

## Step 19: Open Order Management

Log in using the **Route Planner** role.

Navigate to **Order Management**.

Depending on the order type:

- Select **Assigned** for Scheduled Orders.
- Select **Unassigned** for Ad-Hoc Orders.

Apply the required delivery date filter to locate the order.

![Step 19](/img/non-cash-item/step-19.jpeg)

---

## Step 20: Move Orders to a Run

Select the required order(s).

Click **Move to Runs** to assign the selected orders to a delivery run.

![Step 20](/img/non-cash-item/step-20.jpeg)

---

## Step 21: Confirm the Assignment

Review the selected orders and confirm the assignment.

:::tip

Multiple orders can be assigned to the same run when appropriate.

:::

![Step 21](/img/non-cash-item/step-21.jpeg)

---

## Step 22: Open Run Assignment

Navigate to **Run Assignment**.

Select the newly created run to view the assigned orders.

![Step 22](/img/non-cash-item/step-22.jpeg)

---

## Step 23: Create the Runsheet

Move the selected run to a Runsheet.

Provide the following information:

- Service Date
- Assigned User
- Vehicle

Click **Save** to create the Runsheet.

![Step 23](/img/non-cash-item/step-23.jpeg)

---

## Step 24: View the Runsheet

Navigate to **Runsheet** to review the newly created delivery run.

From this screen you can:

- View assigned orders
- Edit the Runsheet
- Monitor delivery progress

![Step 24](/img/non-cash-item/step-24.jpeg)

---

## Step 25: View Order Status

Select the **Status** icon to monitor the current progress of the order throughout the delivery lifecycle.

![Step 25](/img/non-cash-item/step-25.jpeg)

---

## Summary

The Route Planner process enables you to:

- Locate submitted Non Cash Item Orders.
- Assign one or more orders to a delivery run.
- Create a Runsheet for delivery.
- Allocate a service date, vehicle, and driver.
- Monitor the delivery status of assigned orders.

:::tip

After the Runsheet has been created, the assigned delivery team can proceed with the delivery according to the scheduled service date.

:::
