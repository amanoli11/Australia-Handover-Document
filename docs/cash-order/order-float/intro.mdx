# Float Order

:::tip
This guide explains how to create, cancel, and assign a Float Order using Authentic Online.
:::

Float Orders allow customers to request pre-configured cash float profiles for delivery to a selected location.

The complete workflow consists of three stages:

```mermaid
flowchart LR
A[Customer Portal] --> B[Create Float Order]
B --> C[View Confirmation]
C --> D[Crew Sign Off]
D --> E[Route Planner]
E --> F[Assign to Runsheet]
